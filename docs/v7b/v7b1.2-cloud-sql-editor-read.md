## Step 2: Governance Classification Query

Run this to classify the canary's governance status:

```sql
-- v7B.1.2: Governance classification of canary row
SELECT 
    id,
    content,
    metadata,
    metadata->>'source' AS source,
    metadata->>'version' AS version,
    metadata->>'seal' AS seal,
    (metadata->>'isGovernedState')::boolean AS is_governed_state,
    (metadata->>'containsTradeOrders')::boolean AS contains_trade_orders,
    (metadata->>'notExecutionAuthority')::boolean AS not_execution_authority,
    CASE 
        WHEN (metadata->>'isGovernedState')::boolean = true THEN '⚠️ GOVERNED_STATE'
        WHEN (metadata->>'containsTradeOrders')::boolean = true THEN '⚠️ TRADE_ORDERS'
        WHEN (metadata->>'notExecutionAuthority')::boolean = false THEN '⚠️ CLAIMS_EXEC_AUTH'
        WHEN metadata->>'source' IS NULL THEN '⚠️ MISSING_SOURCE'
        ELSE '✅ SAFE — advisory context only'
    END AS governance_status
FROM public.memories
WHERE id = 'd4c9812b-7455-41bb-b382-15e65b1c3ff4';
```

**Expected:** `governance_status` = `✅ SAFE — advisory context only`

**Screenshot this result.**

## Step 3: Scan for Governed-State Rows

Run this to check if any memory rows claim governed state:

```sql
-- v7B.1.2: Scan for governed-state or trade-order memory
SELECT 
    id,
    content,
    metadata,
    CASE 
        WHEN (metadata->>'isGovernedState')::boolean = true THEN '⚠️ GOVERNED_STATE'
        WHEN (metadata->>'containsTradeOrders')::boolean = true THEN '⚠️ TRADE_ORDERS'
        WHEN (metadata->>'notExecutionAuthority')::boolean = false THEN '⚠️ CLAIMS_EXEC_AUTH'
        ELSE '✅ Safe'
    END AS governance_status
FROM public.memories
WHERE (metadata->>'isGovernedState')::boolean = true
   OR (metadata->>'containsTradeOrders')::boolean = true
   OR (metadata->>'notExecutionAuthority')::boolean = false
LIMIT 50;
```

**Expected:** Either zero rows, or rows clearly flagged with governance warnings.

**Screenshot this result.**

## Step 4: Row Count + Table Health

```sql
-- v7B.1.2: Memory table health check
SELECT 
    COUNT(*) AS total_rows,
    COUNT(*) FILTER (WHERE content LIKE 'v7B.1.1-canary%') AS canary_rows,
    COUNT(*) FILTER (WHERE metadata->>'source' = 'v7b1.1-canary') AS v7b11_rows,
    COUNT(*) FILTER (WHERE (metadata->>'isGovernedState')::boolean = true) AS governed_rows,
    COUNT(*) FILTER (WHERE (metadata->>'containsTradeOrders')::boolean = true) AS trade_rows,
    MAX(created_at) AS latest_row
FROM public.memories;
```

**Expected:** Total rows, canary row count, and any governed/trade rows flagged.
